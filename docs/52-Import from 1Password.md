# How to migrate from 1Password to PassHub.net

_Step-by-step instructions_

To migrate your data from your 1Password account to PassHub.net, follow the four steps below.


## Export 1Password Data

**Step 1.** In the desktop application (Windows shown), open the three-dot menu and select the "Export" item:

(On Mac, use File > Export > Your Vault.)

![1Password Desktop Application](/img/migration/1pass_export.jpg)


**Step 2.** In the dialog, select the **1PUX** format, enter your password, and click "Export Data."

![1Password Export Dialog](/img/migration/1pass_export_pux.jpg)

## Import 1PUX file to PassHub

**Step 3.** Open PassHub.net in your browser, click the account icon, and select "Import."

![Passhub import menu item](/img/migration/passhub_import.jpg)

**Step 4.** In the import dialog, browse to and select the exported 1Password **1PUX** file, then click "Import."

![Passhub import dialog](/img/migration/passhub_import_dialog.jpg)


The new safe will be created. You can then rename it to something more readable.

![Imported safe](/img/migration/imported_1pug_safe.jpg)


## Delete the 1PUX file

Now that your data is safely stored in PassHub.net, we highly recommend you delete the **1PUX** file, as it contains unencrypted sensitive data.
